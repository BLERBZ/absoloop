## Palette (infinity ribbon)

| Stop | RGB | Hex-ish role |
|---|---|---|
| Crossing green | `40, 205, 120` | success / integrity |
| Lime → gold | `210, 245, 20` → `255, 190, 15` | attention / spend |
| Orange → pink | `255, 145, 20` → `255, 25, 95` | warn / block |
| Cyan → blue | `0, 205, 225` → `0, 105, 255` | accent / links |
| Magenta | `210, 0, 165` | highlight |

Backgrounds for dark surfaces: near-black `#0c1118`–`#121a24`, not flat pure black in long-form docs when readability matters.

## Voice

Short. Evidence over narrative. Loops are bounded. Critics are adversarial.
No hype about “autonomous AGI” — Absoloop is an orchestrator with receipts.
